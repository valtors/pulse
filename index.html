connect everything. your ai does the rest.

github install

what it does

1

connect your services

github. gmail. calendar. slack. notion. one by one. your tokens stay on your machine.

2

ask what you missed

pulse reads everything you ignored. 247 unread emails. 38 github notifications. today's calendar. it tells you what matters and what doesn't.

3

tell it to do things

"reply to the urgent ones." "star that repo." "remind me about this on friday." pulse does it. or tells you it can't yet.

4

it remembers you

what you care about. what you don't. what you asked last time. persistent memory. local. yours.

install

# install
go install github.com/valtors/pulse/cmd/pulse@latest

# run with ai (any openai-compatible api)
PULSE_LLM_KEY=sk-xxx pulse

# or without ai (raw data only)
pulse

# open http://localhost:9090

api

# connect github
curl -X POST localhost:9090/connect \
  -d '{"service":"github","token":"ghp_xxx"}'

# ask
curl -X POST localhost:9090/ask \
  -d '{"input":"what did i miss"}'

# memory
curl localhost:9090/memory

services

github gmail calendar slack (soon) notion (soon) discord (soon)

why

hermes and openclaw are agent frameworks for developers. pulse is a product for people. you don't write system prompts. you don't configure yaml. you connect your accounts and ask a question. the ai does the rest. your data stays on your machine. no cloud. no account. no subscription.
